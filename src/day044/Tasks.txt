Task 1:
Klavyeden girilen 3 adet tam sayının ortalamasını hesaplayan Java programını yazınız.

Talimatlar:
- Kullanıcı print ile bilgilendirilecektir.
- Scanner ile kalvyeden veri okunacaktır.
- veri okuma işi for döngüsü içerisinde yapılacaktır.
- Okunan sayı toplam değişkenine ilave edilecektir.
- Döngü bitiminde ortalama hesaplanıp, ekranda gösterilecktir.


Task 2:
Parola kontrolü yapan Java programını yazınız.

Talimatlar:
- Kullanıcıdan yeni parolasını girmesi istenilecektir.
- Onay için aynı parolayı tekrar girmesi istenilecektir.
- Girilen her iki değeri karşılaştırıp boolean değer döndüren
checkPassword(password,confirmation) metodu yazılacaktır.
- Eğer metot true döndürürse ekrana "Parola onaylandı" mesajı yazdırılacaktır.
- Eğer metot false döndürürse kırmızı şekilde "Eşleşme sağlanamadı" mesajı yazdırılacaktır.

Task 3:
Verilen bir dizi içerisindeki belirli bir değeri bulan Java programını yazınız.

Talimatlar:
- int findValue(dizi,değer) şeklinde bir metot tasarlanacaktır.
- Metotta bir döngü kullanıarak değer dizi içerisinde aranacaktır.
- Eğer bulunursa ilgili index numarası döndürülecektir.
- Eğer bulunamazsa -1 değeri döndürülecektir.
- Metot çağrıldığında -1'den farklı bir değer dönmüşse "Aradığınız değerin index numarası: 5"
şeklinde bir mesaj yazdırılacaktır.
- Eğer -1 dönmüş ise Kırmızı şekilde "Aradığınız değer bulunamadı" mesajı yazdırılacaktır.
